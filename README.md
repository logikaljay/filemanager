# File Manager for Node
## Requirements
* node
* imagemagick

## NPM install
* express - 3.x
* imagemagick - ~0.1.3
* node-uuid - ~1.4.1
* fs.extra - ~1.2.1

## Plans
* container like storage that issues a GUID when creating to the creator
* all api requests will require this GUID to access any of the files
